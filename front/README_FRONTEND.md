# Pokemon Bytes

**Pokemon Bytes** es un proyecto desarrollado con [Svelte](https://svelte.dev/). Está diseñado principalmente para ser ejecutado en sistemas Linux, y las instrucciones están orientadas a este entorno. A continuación, encontrarás las instrucciones necesarias para configurarlo y ejecutarlo correctamente.


## Requisitos previos
Asegurate de tener instalados los siguientes componentes:

- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/)
- [Git](https://git-scm.com/)

## Instrucciones

1. **Instala las dependecias:**

    ```bash
    npm install
    ```

2. **Ejecutar el servidor de desarrollo:**
    ```bash
    npm run dev
    ```
